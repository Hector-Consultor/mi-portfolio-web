# Project Brief

## Nombre del proyecto
`mi-primera-web` — Portfolio profesional de Héctor González

## Objetivo
Crear y mantener un portfolio web profesional que funcione como:
1. Tarjeta de presentación digital
2. Escaparate de servicios y casos de éxito
3. Canal de captación de clientes freelance

## Alcance
- Sitio web estático de una sola página (SPA)
- Sin backend, sin base de datos, sin autenticación
- Deploy en Netlify con dominio propio implícito (hgautomation.netlify.app)
- Mantenimiento continuo: actualizar contenido, agregar proyectos reales, mejoras de UX

## Fuera del alcance
- Blog o CMS
- Panel de administración
- Formulario de contacto con backend (pendiente como mejora futura)
- E-commerce o pagos

## Criterios de éxito
- [x] Sitio live y accesible 24/7
- [x] Diseño profesional, oscuro, moderno
- [x] Responsive en móvil y desktop
- [x] Carga rápida (sin dependencias externas)
- [x] Información de contacto clara y accesible
- [ ] Casos de éxito con proyectos reales documentados
- [ ] Formulario de contacto funcional

## Restricciones
- Sin frameworks JS (React, Vue, etc.) — solo vanilla
- Sin librerías CSS (Bootstrap, Tailwind) — solo CSS propio
- Archivo único (index.html) — sin bundler ni build step
- Deploy automático via GitHub → Netlify

## Stakeholder
Héctor González — propietario, desarrollador y cliente final.
